# Career-Twin-AI
🚀 CareerTwin AI is an AI-powered career coach that analyzes resumes, identifies skill gaps, and generates personalized learning roadmaps to help users achieve their dream careers. Built with Next.js, Tailwind CSS, OpenAI, and modern web technologies for the OpenAI x Devpost Hackathon.

### Your AI-powered Career Coach

CareerTwin AI analyzes resumes, identifies skill gaps, and generates personalized career roadmaps using OpenAI. Designed to help students and professionals accelerate their career growth with AI-powered insights.

## ✨ Features
- 📄 AI Resume Analysis
- 🎯 Skill Gap Detection
- 🛣️ Personalized Career Roadmap
- 🤖 AI Career Coach
- 📊 Career Progress Dashboard
- 💼 Project Recommendations

## 🛠️ Tech Stack
- Next.js
- React
- Tailwind CSS
- TypeScript
- OpenAI API
- Supabase
- Vercel

## 🎯 Built For
OpenAI x Devpost Hackathon 2026
